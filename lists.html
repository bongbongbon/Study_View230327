목록을 만들 때 사용하는 태그

크게 3가지 목록이 있고, 기본적인 문법과 사용방법 익히기

순서가 없는 목록

Unordered List를 줄여서 ul 이라고 하는 명령어로 표현하고, 내부 항목은 List Item이라는 의미를 가진 li 태그명령어로 표현한다.

내가 좋아하는 것 (음식)

  • 김치
  • 피자
  • 삼겹살

순서가 있는 목록

Ordered List를 줄여서 ol 이라고 하는 명령어로 표현

올해 가고싶은 여행지(순위)

  1. 베트남
  2. 파리
  3. 이스탄불

순서가 있는 목록

Definition List를 줄여서 dl 이라는 태그로 표현하고, 내부항목은 용어를 뜻하는 Definition Term, dt태그그리고 내용은 Definition Description 즉, dd태그로 표현한다.

일반상식 백과사전

자동차
내연기관으로 움직이는 이동수단으로 사람과 물자를 옮기는데 사용한다.
비행기
이동수단, 운동수단으로 하늘을 나는 기계장치
스쿠터
새롭게 등장한 이동수단, 면허가 없어도 이용할 수 있고 시간당 1000원의 비용이 평균

라면을 맛있게 조리하는 방법

끓는 물 550ml를 준비하고, 스프를 먼저 넣고 그 다음에 면을 넣고 그리고 3분을 더 가열한 뒤, 면을 이리저리 뒤집어서 꼬들꼬들하게 만들고 된장 1T를 넣는다(된장라면?)

  1. 냄비에 물 550ml를 준비한다
  2. 펄펄 끓을때 까지 3분 가열한다.
  3. 물이 끓으면 스프를 넣는다.
  4. 잠시 기다렸다가 면을 넣는다.
  5. ...맛있게 먹는다...
